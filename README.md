## üìö About This Wiki

This repository contains the complete user documentation for [BrewOS](https://github.com/brewos-io/firmware), an open-source control system for espresso machines. The wiki provides:

- **Installation guides** - Step-by-step hardware and firmware installation
- **User manuals** - Complete guide to using BrewOS
- **Feature documentation** - Detailed documentation for all features
- **Machine compatibility** - Hardware compatibility lists and machine-specific guides
- **Troubleshooting** - Common issues and solutions
- **Advanced topics** - Custom configurations and API reference
