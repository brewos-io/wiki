## Installation Overview

The BrewOS installation process consists of three main steps:

1. **Hardware Installation** - Install the control board, sensors, and wiring
2. **Firmware Flashing** - Flash firmware to both Pico and ESP32
3. **Initial Setup** - Configure WiFi and complete first-run wizard

## Prerequisites

Before starting installation:

- ✅ Verify machine compatibility (see [System Requirements](../01-Getting-Started/System-Requirements.md))
- ✅ Have all required hardware components
- ✅ Understand electrical safety requirements
- ✅ Have necessary tools and equipment

## Safety First

⚠️ **CRITICAL: MAINS VOLTAGE WORK**

This installation involves 100-240V AC mains electricity. Improper handling can result in death or serious injury.

**Only qualified individuals should perform this installation.**

- Always disconnect power before making changes
- Use isolation transformers during development
- Never work alone on energized equipment
- Follow all local electrical codes and regulations
