## Feature Categories

### Core Features
- **Temperature Control** - Precise PID control
- **Brewing Features** - Shot timer, pre-infusion, brew by weight
- **Statistics** - Shot history and analytics

### Automation
- **Scheduling** - Auto-on/off schedules
- **Eco Mode** - Energy-saving idle mode

### Connectivity
- **WiFi Connectivity** - Local network access
- **Cloud Access** - Remote access
- **Home Assistant** - Smart home integration
- **MQTT** - Automation protocol

### Monitoring
- **Power Metering** - Energy consumption
- **Push Notifications** - Alerts and notifications
- **Statistics** - Data tracking

### Maintenance
- **OTA Updates** - Wireless firmware updates

## Getting Started with Features

1. **Core features** are available immediately after installation
2. **Connectivity features** require network configuration
3. **Integrations** require additional setup (see individual guides)
4. **Optional features** require additional hardware (power meter, scale)

## Feature Status

All features listed here are implemented and available. Some features may require:
- Additional hardware (power meter, scale)
- Network configuration (cloud, MQTT)
- Third-party services (Home Assistant)

See individual feature pages for requirements and setup instructions.
