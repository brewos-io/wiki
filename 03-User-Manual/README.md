## Getting Started

If you've just completed installation:

1. **Access the web interface** - Connect to your machine via WiFi
2. **Familiarize yourself** - Read [Web Interface Guide](Web-Interface-Guide.md)
3. **Learn the dashboard** - See [Dashboard Overview](Dashboard-Overview.md)
4. **Control your machine** - Follow [Machine Control](Machine-Control.md)
5. **Configure settings** - Review [Settings Overview](Settings-Overview.md)

## Quick Reference

### Accessing the Interface

- **Local network**: `http://brewos.local` or ESP32 IP address
- **Cloud access**: `https://cloud.brewos.io` (if configured)
- **Install as PWA**: Use browser install option for app-like experience

### Main Pages

- **Dashboard** - Main control and monitoring
- **Brewing** - Shot timer and brew controls
- **Statistics** - Shot history and graphs
- **Schedules** - Auto-on/off scheduling
- **Settings** - Configuration options
- **Logs** - System diagnostics
